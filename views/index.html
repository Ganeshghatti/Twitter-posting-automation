Getting Started

  1. Make sure your CONSUMER_KEY and CONSUMER_KEY_SECRET are set in .env
  2. Add {{CALLBACK_URL}} to your app's callback URLs in the X Developer Portal
  3. Click the button below to start the OAuth flow and get your access tokens
  4. Copy the tokens to your .env file
  5. Install and start Ollama: ollama serve then ollama pull llama3.2:3b
  6. Set ENABLE_SCHEDULER=true in .env to enable automated posting
  7. Restart the server - tweets will post automatically!
